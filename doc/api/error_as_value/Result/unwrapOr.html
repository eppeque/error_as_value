Returns the contained ok value or a provided defaultValue.

Arguments passed to unwrapOr are eagerly evaluated; if you are passing the result of a function call, it is recommended to use Result.unwrapOrElse, which is lazily evaluated.

Examples

const default = 2;
final Result<int, String> x = Result.ok(9);
print(result.unwrapOr(default)); // 9

final Result<int, String> y = Result.err("error");
print(y.unwrapOr(default)); // 2

Implementation

T unwrapOr(T defaultValue) {
  if (_value != null) return _value;

  return defaultValue;
}
